# BioBench Legacy V2

This folder contains the older production BioBench v2 row definitions.

Files:

- `biobench_v2_manifest_clean.csv`: clean default production row set, 630 rows.
- `biobench_v2_manifest_unfiltered.csv`: audit-inclusive row set, 805 rows.
- `biobench_v2_ligand_modules.csv`: curated ligand module catalog, 23 modules.
- `biobench_v2_summary.json`: generated row/module counts.

These are benchmark row definitions and module annotations, not model outputs.
