## Method

Gene shuffles preserve response-conversion panel sizes and signs. For GE5
backfill rows, the control also randomizes the MOA-axis panel with the same
number of up/down genes as the mapped MOA prior. The frozen selected variant is
then applied and the patient rows are re-scored through the universal response
axis contract.

Hard donors replace the source response probability, conversion features, and
MOA-axis features with same-patient wrong-drug donors before re-scoring the
target therapy context.
