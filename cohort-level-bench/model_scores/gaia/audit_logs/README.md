# Gaia Audit Logs

These are compact audit artifacts for the active/default Gaia cohort score.

Included:

- `active_default_input_comparability_required_inputs.csv`: required input
  completeness and ranges.
- `active_default_input_comparability_axis_policy_source.csv`: cohort-level
  split of axis policies and score means.
- `active_default_input_comparability_moa_completeness.csv`: explicit MOA
  feature completeness by cohort and prediction source.
- `active_default_input_comparability_prevalence_source.csv`: prevalence source
  and semantic provenance.
- `active_default_input_comparability_nominal_sample_source_split_sanitized.csv`:
  nominal sample source split aggregated without raw patient identifiers.
- `active_default_input_comparability_report_excerpt.md`: aggregate audit
  report excerpt before raw nominal-sample rows.
- `gene_shuffle_summary.csv`, `hard_donor_summary.csv`, and
  `susceptibility_report.md`: retained susceptibility controls.

The full 1,078-row patient probability table and raw nominal-sample audit table
are not part of the public release folder.
